# Portafolio

- se realizo la maquetacion de un portafolio
- tecnologias usadas:
   Html, Css, JavaScript
   
   
 ## Usos
 
 se puede replicar este portafolio para crear uno personalizado y de esta forma hacerle saber a los reclutadores las capacidades y conocimientos 
 que manejamos, brindando la informacion mas relevante y dejando tambien la forma de contactarnos.
 
 ## Herramientas utilizadas
 
 - Agregar bordes personalizados a nuestras imagenes 
 
 link https://9elements.github.io/fancy-border-radius/#40.35.27.33--.
 
 - Agregar iconos, los cuales se agregan usando un script en el head del html del proyecto:
 
 link https://fontawesome.com/kits/eb8ab39ed3/setup
 
 ## Modo Oscuro
 
 - se agrega el modo oscuro al proyecto con el uso de JavaScript de la siguiente manera:
 
   <script>
        const darkMode = document.querySelector('.switch');
        darkMode.addEventListener('click', ()=>{
            document.body.classList.toggle('darkmode');
        })
    </script>
